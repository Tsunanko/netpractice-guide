## 🎯 絶対守るルール 5 か条

### 1. スイッチ配下は全員同じサブネット

```
[A] ─┐
[B] ─┼ Switch → 全員同じ IP 帯 + 同じマスク
[C] ─┘
```

### 2. ルータの各口は別サブネット

```
[Router]
  口 R1 → サブネット A
  口 R2 → サブネット B
```

### 3. ゲートウェイは自分と同じサブネット内

```
Host (192.168.1.10/24)
  Gateway ∈ 192.168.1.0/24 の範囲  ← 必須
```

### 4. 通信は双方向

```
A → B の route と B → A の route は 独立
どちらか欠けると通信失敗
```

### 5. ルーティングテーブルは **具体 → default** の順

```
route: 10.0.0.0/8 → gate1
route: 172.16.0.0/12 → gate2
route: default → gate3   ← 必ず最後
```

## 🔎 IP → ブロック帰属の計算手順（3 秒）

### 手順

1. **ブロックサイズ** = 256 − マスク最後の桁
2. **IP の最後の桁** ÷ ブロックサイズ を切り捨て
3. **× ブロックサイズ** = ブロック先頭

### 例: `192.168.1.222 / 27`

```
ブロックサイズ: 256 − 224 = 32
222 ÷ 32 = 6.9 → 6
6 × 32 = 192

→ .192/27 ブロック
→ ネットワーク: .192
→ ホスト: .193 〜 .222
→ ブロードキャスト: .223
```

## 📋 レベル別の覚えポイント

| Level | 核心 |
|:-:|:---|
| 1 | 直結 = 同じサブネット |
| 2 | **マスクは 1 連続の後 0 連続** でなければ無効 |
| 3 | スイッチ配下は全員同じ |
| 4 | ルータ各口は別、**他口と被らないブロック** を選ぶ |
| 5 | default route + gate はルータの近い口 |
| 6 | **Internet 側の route** を広げる（帰り道） |
| 7 | /24 を **/26 で 3 分割** |
| 8 | **隣接ブロックを /27 で集約** |
| 9 | 固定値から逆算、**Internet に複数 route** |
| 10 | **Ir1 = `0.0.0.0/0`** で帰り全部まとめる |
